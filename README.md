## 头条快讯系统

> [点我下载源码或在线试用](https://www.notmaker.com/detail/773bfba029a945e3847bdbb7d1fc48f4/ghb20250810) 

> 支持定制、讲解、修改、远程调试

### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**用户端功能：**
1. 登录
2. 注册
3. 首页
4. 文章
5. 个人中心
6. 后台管理（首页、个人中心、文章管理）




**自媒体人端功能：**
1. 登录
2. 注册
3. 首页
4. 文章
5. 个人中心
6. 后台管理（首页、个人中心、文章管理）


**管理员功能：**
1. 登录
2. 首页
3. 个人中心（修改密码、个人信息）
4. 基础数据管理
5. 文章管理（文章管理、文章留言管理、文章收藏管理）
6. 用户管理
7. 自媒体人管理
8. 轮播图信息
